Développement d'une application web de type SaaS, pour les hypnothérapeutes.

## Technologies

Ce projet de Saas (Software as a Service) est né d'une discussion avec [FanHypnose]("/projects/fanhypnose") : aucun logiciel n'est à disposition des hypnothérapeutes pour leur permettre de gérer leur entreprise facilement.

J'ai donc décidé de me lancer dans le développement de cette application, qui est encore en cours de développement.
